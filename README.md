# 🚀 Bootstrap Mastery: Laboratório de Práticas

Bem-vindo ao repositório de **Exemplos Práticos de Bootstrap**. Este espaço foi projetado para servir como um guia de referência dinâmica, reunindo os conceitos fundamentais do framework aplicados na construção de interfaces ágeis e responsivas. Se você está aprendendo sobre o sistema de grids ou implementando componentes avançados, aqui encontrará a base para acelerar o desenvolvimento front-end. 💻📱

---

## 💡 O que é o Bootstrap?

O Bootstrap é um dos frameworks CSS mais populares do mundo, focado em facilitar o desenvolvimento de interfaces responsivas e mobile-first. Com ele, utilizamos classes utilitárias pré-definidas e componentes robustos (como modais, carrosséis e navbars) para criar layouts profissionais de forma rápida, mantendo a consistência visual e a compatibilidade entre diferentes navegadores.

---

## 📂 Organização do Aprendizado

Os exemplos estão segmentados para facilitar a navegação e o estudo dirigido das turmas de Desenvolvimento de Sistemas no Senai/Sesi, evoluindo desde a instalação até projetos completos:

### ⚙️ Instalação e Guias Básicos
* **Guia Completo e Ícones**: Introdução ao framework e uso do Bootstrap Icons (`ex0_guia-completo-bootstrap`, `ex0a_guia-icones-bootstrap`).
* **Métodos de Instalação**: Via CDN (`ex1_instalacao-cdn`), arquivos locais (`ex1a_instalacao_arquivos`) e NPM (`ex1b__instalacao_npm`).

### 📐 Layout e Sistema de Grid
* **Sistema de Grid**: Estruturação de páginas responsivas em linhas e colunas (`ex2_sistema-grid`).
* **Layout Completo**: Construção de estruturas de página de ponta a ponta (`ex10_layout-completo`).

### 🎨 Componentes Visuais (UI)
* **Botões e Cards**: Elementos de interação e exibição de conteúdo (`ex3_botoes`, `ex5_cards`).
* **Navegação**: Implementação de barras de navegação responsivas (`ex6_navbar`).
* **Elementos Interativos**: Janelas sobrepostas e slides de imagens (`ex7_modal`, `ex8_caroussel`).
* **Feedback e Progresso**: Alertas e barras de progresso com integração JS (`ex9_alerts_progress_sub-js`).
* **Formulários**: Estilização de inputs e formulários (`ex4_formulario-sub-css`).

### 🚀 Projetos Práticos
* **Projeto Art**: Aplicação prática dos conceitos (`ex11_projeto_art`).
* **Projeto Perfect Watch**: Landing page temática (`ex12_projeto-perfectwatch`).

## 👨‍💻 Como Explorar este Repositório

Siga os passos abaixo para rodar os projetos em sua máquina local:

1.  **Clone este laboratório:**
    ```bash
    git clone [https://github.com/seu-usuario/bootstrap-exemplos.git](https://github.com/seu-usuario/bootstrap-exemplos.git)
    ```
  
2.  **Acesse a pasta do projeto:**
    ```bash
    cd bootstrap-exemplos
    ```

3.  **Execução:**
    Abra qualquer arquivo `.html` das pastas de exercícios diretamente no navegador. 

> 💡 **Dica de mestre:** No VS Code, utilize a extensão **Live Server** para visualizar as alterações em tempo real sem precisar atualizar a página manualmente.
